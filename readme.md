# Solla Dental Clinic Patient Records System

A website that stores a sample dental clinic's patient records. This site was commissioned by a Dental Medicine student.

## Setup

### Installation

```
git clone https://github.com/axe1a/DENTAL_CLINIC_PATIENT_RECORDS.git
cd DENTAL_CLINIC_PATIENT_RECORDS

```

### Usage

To start the website and database, just run `start.bat`. It will download and setup PHP on your first run.

<br>

## Setup (for development)

### Prerequisites

- [Node.js](https://nodejs.org/en)

### Installation

```
git clone https://github.com/axe1a/DENTAL_CLINIC_PATIENT_RECORDS.git
cd DENTAL_CLINIC_PATIENT_RECORDS

```

Install dependencies using `npm install`. <br>
To edit Tailwind CSS classes, run `npx @tailwindcss/cli -i ./app/imports/importTW.css -o ./app/styles/tailwindStyles.css --watch`

### Setup

To start the website and database, just run `start.bat`
